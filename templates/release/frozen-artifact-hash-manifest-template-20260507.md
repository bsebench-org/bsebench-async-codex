## Completion State

Choose exactly one state.

| State | Meaning | Selected |
| --- | --- | --- |
| `complete` | Every required artifact exists, has a stable identity, and has a recorded hash. | `<yes/no>` |
| `blocked_missing_artifact` | At least one required artifact is absent. | `<yes/no>` |
| `blocked_unhashable_artifact` | At least one artifact exists but cannot be hashed reproducibly. | `<yes/no>` |
| `blocked_dirty_source` | At least one source repository has uncommitted or unpushed changes. | `<yes/no>` |

| `blocked_validator_missing` | No independent manifest validator or replay command exists yet. | `<yes/no>` |
